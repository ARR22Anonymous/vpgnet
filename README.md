# Introduction
This is the code for MVPNet.This code is only used for research purposes.


# Dependencies
- PyTorch version >= 1.5.0
- Python version >= 3.6

```
git clone https://github.com/pytorch/fairseq.git
cd fairseq 
pip install --editable ./
``` 

# Running Commands
## data preprocessing
```
bash preprocess_pipline.sh
```

## train
```
bash run_train.sh
```

## predict
```
bash run_predict.sh
```

## eval
```
bash run_eval.sh
```
